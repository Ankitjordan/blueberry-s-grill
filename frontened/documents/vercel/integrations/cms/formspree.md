# Vercel Formspree Integration

Ask AI about this page

Last updated March 4, 2025

Formspree is a form backend platform that handles form submissions on static websites. It allows developers to collect and manage form data without needing a server.

## [Getting started](https://vercel.com/docs/integrations/cms/formspree#getting-started)[](https://vercel.com/docs/integrations/cms/formspree#getting-started)

To get started with Formspree on Vercel, follow the steps below to install the integration:

1.   ### [Install the Vercel CLI](https://vercel.com/docs/integrations/cms/formspree#install-the-vercel-cli)[](https://vercel.com/docs/integrations/cms/formspree#install-the-vercel-cli) 
To pull in environment variables from Formspree to your Vercel project, you need to install the [Vercel CLI](https://vercel.com/docs/cli). Run the following command in your terminal:

 Terminal ![Image 3](https://7nyt0uhk7sse4zvn.public.blob.vercel-storage.com/docs-assets/static/topics/icons/pnpm.svg)        `pnpm i -g vercel@latest`     `yarn global add vercel@latest`     `npm i -g vercel@latest`     `bun add -g vercel@latest`   
2.   ### [Install your CMS integration](https://vercel.com/docs/integrations/cms/formspree#install-your-cms-integration)[](https://vercel.com/docs/integrations/cms/formspree#install-your-cms-integration) 
Navigate to the [Formspree integration](https://vercel.com/integrations/formspree) and follow the steps to install the integration.

3.   ### [Pull in environment variables](https://vercel.com/docs/integrations/cms/formspree#pull-in-environment-variables)[](https://vercel.com/docs/integrations/cms/formspree#pull-in-environment-variables) 
Once you've installed the Formspree integration, you can pull in environment variables from Formspree to your Vercel project. In your terminal, run:

   `vercel env pull` 

See your installed CMSs documentation for next steps on how to use the integration.
